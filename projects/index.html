  • Header Field Extraction
    • Working with members of the Information Extraction and Synthesis Laboratory at UMass Amherst, directed by Prof. Andrew McCallum, on a new annotated dataset of scientific research papers from bioRxiv in collaboration with Chan Zuckerberg Initiative.
    • I am responsible for developing the pipeline which will parse the data, train the machine learning models on the data and obtain the F1 scores from the test set.


  • Citation Field Extraction
    • Worked with Chan Zuckerberg Initiative as part of an industry mentorship program.
    • Created an automated pipeline that can segment any citation string found in scientific research papers using labels such as author, title, journal, year, etc..
    • Used TensorFlow to implement CRF, LSTM, Dilated-CNN and BiLSTM-CRF to learn from the training data.


  • Wordify: A Reverse Dictionary for Everyone
    • Developed a reverse dictionary which can summarize any phrase to a single word.
    • Trained a LSTM network using pretrained GloVe embeddings as the starting point on the WordNet dataset.
    • The LSTM based method outperformed the baseline methods which were based on the arithmetic properties of the word embeddings.
    • Keras w/ TensorFlow backend was used to design the model.


  • Sudoku Solver Try it out!
    • Developed an AI agent that can solve any Sudoku puzzle provided as an input.
    • Constraint Propagation was used to efficiently reduce the domain of the variables.
    • Minimum Remaining Values heuristic was used to select the most optimal variable for assignment.


  • Tic Tac Toe
    • Developed an AI agent that can play the game of Tic Tac Toe with a human.
    • Minimax algorithm is used to find the best possible move for any configuration, which makes the agent unbeatable.


  • Monte Carlo Image De-noising
    • Applied Monte Carlo Inference to de-noise any noisy image given as an input.
    • Gibbs sampling was used to obtain observations from the CRF model of the image.


  • Motley: Color the Uncolored
    • Developed a system based on CNN that can color a grayscale image given as an input.
    • Developed a baseline method to compare the performance of the CNN.
    • The model was trained on resized images from the CIFAR-10 dataset.
    • Keras w/ TensorFlow backend was used to design the model.


  • Autonomous Navigation and Control of an Omni-Directional Vehicle
    • Implemented path-planning algorithms on a four-wheeled omni drive vehicle using sensor fusion technique.
    • STM-32 was used to drive the motors and Raspberry Pi was used as a CPU.
